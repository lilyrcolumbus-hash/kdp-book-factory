AI Images - Workflow

⚙️ Quick setup (one time) — bookmarklet for almost automatic workflow
  1. Configure OpenArt once: open OpenArt → Create → select Flux 1.1 Pro model + 2:3 aspect + 3 samples. Generate 1 test image. Those settings stay as default while your session is active.
  2. Pin the tab: right-click OpenArt tab → "Pin Tab". That way it won't close between images.
  3. Install the bookmarklet: show bookmarks bar (Cmd+Shift+B), then drag the yellow button below to your bookmarks bar.
